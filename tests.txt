New endpoints
All require Authorization: Bearer <TOKEN>.
Album
POST /api/v1/albums/:id/like → like album
DELETE /api/v1/albums/:id/like → unlike album
GET /api/v1/albums/:id/reactions → { likes, comments }
POST /api/v1/albums/:id/comments body: { content } → create comment
GET /api/v1/albums/:id/comments → list comments with user info
Media (within album)
POST /api/v1/albums/:id/media/:mediaId/like
DELETE /api/v1/albums/:id/media/:mediaId/like
GET /api/v1/albums/:id/media/:mediaId/reactions → { likes, comments }
POST /api/v1/albums/:id/media/:mediaId/comments body: { content }

Albums list: fetch /albums as you do; to display counts, call /albums/:id/reactions and merge {likes, comments}.
Detail modal:
Media likes count: call /albums/:id/media/:mediaId/reactions.
Like toggle: POST or DELETE /albums/:id/media/:mediaId/like.
Comments list: GET /albums/:id/media/:mediaId/comments.
